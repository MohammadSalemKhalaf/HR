Features

✅ Task List with Checkboxes

Each task displays with a clickable checkbox. Users can quickly mark tasks as complete.

<input type="checkbox" :checked="task.completed" @change="toggleTaskCompletion(task)" />

🔢 Task Counter

A badge displays the total number of tasks assigned to the employee.

<span class="rounded-full bg-blue-50">{{ tasks.length }}</span>

📝 Task Description

Each task can have a description that provides more context about what needs to be done.

<p>{{ task.description }}</p>

🎨 Visual Feedback

Completed tasks show a strikethrough effect and gray color for easy visual distinction.

:class="{ 'line-through text-slate-400': task.completed }"

🔄 Real-time Sync

When a task is marked as complete/incomplete, changes are saved to the backend API automatically.

await api.patch(`/tasks/${task.id}`, { status: task.completed ? 'completed' : 'pending' })

Implementation

<section class="rounded-3xl border border-slate-200 bg-white p-6">
  <div class="mb-5 flex items-center justify-between">
    <h2 class="text-lg font-bold text-slate-900">My Tasks</h2>
    <span class="rounded-full bg-blue-50 px-3 py-1">{{ tasks.length }}</span>
  </div>
  
  <div v-if="tasks.length > 0" class="space-y-3">
    <div v-for="task in tasks" :key="task.id" class="flex items-start gap-3 p-3 rounded-2xl bg-slate-50 hover:bg-slate-100">
      <input
        type="checkbox"
        :checked="task.completed"
        @change="toggleTaskCompletion(task)"
        class="w-4 h-4 rounded cursor-pointer accent-blue-600"
      />
      <div class="flex-1">
        <label class="text-sm font-medium" :class="{ 'line-through text-slate-400': task.completed }">
          {{ task.title }}
        </label>
        <p v-if="task.description" class="text-xs text-slate-500 mt-1">{{ task.description }}</p>
      </div>
    </div>
  </div>
  
  <div v-else class="text-center py-6">
    <p class="text-sm text-slate-500">No tasks assigned yet</p>
  </div>
</section>
